## Padding

Apply padding using the `p-{key}` prefix family. Replace `{key}` with any value from the spacing scale above.

| Prefix | CSS Property |
|--------|--------------|
| `p-{key}` | `padding` |
| `px-{key}` | `padding-left`, `padding-right` |
| `py-{key}` | `padding-top`, `padding-bottom` |
| `pt-{key}` | `padding-top` |
| `pr-{key}` | `padding-right` |
| `pb-{key}` | `padding-bottom` |
| `pl-{key}` | `padding-left` |

```html
<!-- Uniform padding -->
<div class="p-4">All sides, 1rem</div>

<!-- Horizontal and vertical padding -->
<button class="px-6 py-3">Button</button>

<!-- Individual sides -->
<div class="pt-8 pb-4 pl-6">
  Custom padding per side
</div>
```

## Margin

Apply margin using the `m-{key}` prefix family. Replace `{key}` with any value from the spacing scale above.

| Prefix | CSS Property |
|--------|--------------|
| `m-{key}` | `margin` |
| `mx-{key}` | `margin-left`, `margin-right` |
| `my-{key}` | `margin-top`, `margin-bottom` |
| `mt-{key}` | `margin-top` |
| `mr-{key}` | `margin-right` |
| `mb-{key}` | `margin-bottom` |
| `ml-{key}` | `margin-left` |

```html
<!-- Center a block element horizontally -->
<div class="mx-auto max-w-2xl">Centered content</div>

<!-- Vertical rhythm -->
<h2 class="mt-8 mb-4">Section heading</h2>
<p class="mb-6">Paragraph text</p>

<!-- Auto margin for push layout -->
<nav class="flex">
  <a href="/">Logo</a>
  <a href="/login" class="ml-auto">Login</a>
</nav>
```

<Note>
  `auto` is only valid for margin utilities, not padding. Use `mx-auto` or `my-auto` to center elements.
</Note>

## Negative Values

Prefixing any spacing class with `-` negates the value. This works for both margin and sizing (inset) utilities.

```html
<!-- Pull an element upward -->
<div class="relative">
  <div class="-mt-4">Pulled up by 1rem (margin-top: -1rem)</div>
</div>

<!-- Overlap a card over the section above it -->
<section class="pb-12 bg-base">
  <!-- content -->
</section>
<div class="-mt-6 mx-4 bg-surface rounded-xl shadow-lg p-6">
  Overlapping card
</div>
```

<Note>
  Negative values are not applied when the spacing value is `0`. `z-0` stays `0`, not `-0`.
</Note>
