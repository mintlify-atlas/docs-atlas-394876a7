## Width

Control the `width` of an element with `w-{key}`. Accepts any key from the [spacing scale](/utilities/spacing#spacing-scale), fractional values, and special keywords.

```html
<div class="w-0">width: 0</div>
<div class="w-px">width: 1px</div>
<div class="w-4">width: 1rem</div>
<div class="w-full">width: 100%</div>
<div class="w-screen">width: 100vh</div>
<div class="w-min">width: min-content</div>
<div class="w-max">width: max-content</div>
<div class="w-fit">width: fit-content</div>
```

### Fractional Widths

Use `w-{n}/{m}` to set percentage-based widths. The value is computed as `n/m * 100%`.

| Class | CSS |
|-------|-----|
| `w-1/2` | `width: 50%` |
| `w-1/3` | `width: 33.333333%` |
| `w-2/3` | `width: 66.666667%` |
| `w-1/4` | `width: 25%` |
| `w-3/4` | `width: 75%` |
| `w-1/5` | `width: 20%` |
| `w-2/5` | `width: 40%` |
| `w-3/5` | `width: 60%` |
| `w-4/5` | `width: 80%` |
| `w-1/6` | `width: 16.666667%` |
| `w-5/6` | `width: 83.333333%` |

## Height

Control the `height` of an element with `h-{key}`. Accepts all the same keys as width, including fractional values.

```html
<div class="h-16">height: 4rem</div>
<div class="h-full">height: 100%</div>
<div class="h-screen">height: 100vh</div>
<div class="h-svh">height: 100svh</div>
<div class="h-dvh">height: 100dvh</div>
<div class="h-1/2">height: 50%</div>
```

---

## Size (Width + Height)

`size-{key}` sets **both** `width` and `height` to the same value in a single utility. Accepts all spacing scale keys and fractional values.

```html
<!-- Square avatar -->
<img class="size-10 rounded-full" src="avatar.png" />

<!-- Icon container -->
<div class="size-6 flex items-center justify-center">
  <svg>...</svg>
</div>

<!-- Half-width and half-height square -->
<div class="size-1/2">width: 50%; height: 50%</div>
```

## Max-Width

Constrain the maximum width of an element with `max-w-{key}`.

| Class | CSS |
|-------|-----|
| `max-w-none` | `max-width: none` |
| `max-w-xs` | `max-width: 20rem` |
| `max-w-sm` | `max-width: 24rem` |
| `max-w-md` | `max-width: 28rem` |
| `max-w-lg` | `max-width: 32rem` |
| `max-w-xl` | `max-width: 36rem` |
| `max-w-2xl` | `max-width: 42rem` |
| `max-w-3xl` | `max-width: 48rem` |
| `max-w-4xl` | `max-width: 56rem` |
| `max-w-5xl` | `max-width: 64rem` |
| `max-w-6xl` | `max-width: 72rem` |
| `max-w-7xl` | `max-width: 80rem` |
| `max-w-full` | `max-width: 100%` |
| `max-w-screen` | `max-width: 100vw` |
| `max-w-min` | `max-width: min-content` |
| `max-w-max` | `max-width: max-content` |
| `max-w-fit` | `max-width: fit-content` |

```html
<!-- Centered page layout with max-width -->
<main class="max-w-5xl mx-auto px-6">
  Page content
</main>

<!-- Constrain a modal dialog -->
<div class="max-w-md w-full bg-surface rounded-xl p-6">
  Modal
</div>

<!-- Prose reading column -->
<article class="max-w-2xl mx-auto">
  Long-form content
</article>
```

## Min-Width

Set a minimum width with `min-w-{key}`. Accepts any key from the spacing scale.

```html
<button class="min-w-24 px-4 py-2">Button</button>
<div class="min-w-full">At least 100% wide</div>
<div class="min-w-0">Can shrink below natural width</div>
```

---

## Min-Height & Max-Height

Control minimum and maximum heights with `min-h-{key}` and `max-h-{key}`. Both accept any key from the spacing scale.

```html
<!-- Full viewport minimum height for a page shell -->
<div class="min-h-screen flex flex-col">
  <header>...</header>
  <main class="flex-1">...</main>
  <footer>...</footer>
</div>

<!-- Scrollable sidebar with max height -->
<aside class="max-h-screen overflow-y-auto">
  Navigation
</aside>

<!-- Card with min height -->
<div class="min-h-48 bg-surface rounded-lg p-4">
  Content area
</div>
```

<Tip>
  A common pattern for full-page layouts: `min-h-screen` on the outer shell ensures the page always fills the viewport, while `flex-1` on `<main>` pushes the footer to the bottom.
</Tip>
