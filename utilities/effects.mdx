## Shadows

Apply `box-shadow` to elements with `shadow` and `shadow-{size}` utilities.

| Class | CSS |
|-------|-----|
| `shadow-sm` | `box-shadow: 0 1px 2px 0 rgba(0,0,0,0.05)` |
| `shadow` | `box-shadow: 0 1px 3px 0 rgba(0,0,0,0.1), 0 1px 2px -1px rgba(0,0,0,0.1)` |
| `shadow-md` | `box-shadow: 0 4px 6px -1px rgba(0,0,0,0.1), 0 2px 4px -2px rgba(0,0,0,0.1)` |
| `shadow-lg` | `box-shadow: 0 10px 15px -3px rgba(0,0,0,0.1), 0 4px 6px -4px rgba(0,0,0,0.1)` |
| `shadow-xl` | `box-shadow: 0 20px 25px -5px rgba(0,0,0,0.1), 0 8px 10px -6px rgba(0,0,0,0.1)` |
| `shadow-2xl` | `box-shadow: 0 25px 50px -12px rgba(0,0,0,0.25)` |
| `shadow-none` | `box-shadow: none` |

```html
<div class="shadow-sm rounded-lg p-4">Subtle shadow</div>
<div class="shadow rounded-lg p-4">Default shadow</div>
<div class="shadow-xl rounded-xl p-6">Elevated modal-style card</div>
```

## Transitions

Smooth state changes with transition utilities. All transition classes default to `150ms` duration and the `cubic-bezier(0.4, 0, 0.2, 1)` easing function.

### Transition Properties

| Class | `transition-property` |
|-------|----------------------|
| `transition` | `color, background-color, border-color, text-decoration-color, fill, stroke, opacity, box-shadow, transform, filter, backdrop-filter` |
| `transition-all` | `all` |
| `transition-colors` | `color, background-color, border-color, text-decoration-color, fill, stroke` |
| `transition-opacity` | `opacity` |
| `transition-shadow` | `box-shadow` |
| `transition-transform` | `transform` |
| `transition-none` | `none` |

### Duration

Override the transition duration with `duration-{ms}` where `{ms}` is any number of milliseconds.

```html
<button class="transition-colors duration-300">Slow color fade</button>
<div class="transition-opacity duration-75">Fast fade</div>
```

### Easing

| Class | `transition-timing-function` |
|-------|-----------------------------|
| `ease-linear` | `linear` |
| `ease-in` | `cubic-bezier(0.4, 0, 1, 1)` |
| `ease-out` | `cubic-bezier(0, 0, 0.2, 1)` |
| `ease-in-out` | `cubic-bezier(0.4, 0, 0.2, 1)` |

```html
<!-- Hover color transition -->
<button class="bg-accent text-white transition-colors duration-150 ease-in-out">
  Hover me
</button>

<!-- Fade in/out an element -->
<div class="opacity-0 transition-opacity duration-200 ease-out">
  Fade target
</div>

<!-- Shadow on hover -->
<div class="shadow transition-shadow duration-200 ease-in-out">
  Card with hover shadow
</div>
```

```html
<!-- Hidden but accessible label for screen readers -->

  <svg aria-hidden="true">...</svg>
  <span class="sr-only">Close dialog</span>
</button>

<!-- Prevent text selection on a UI element -->
<div class="select-none">
  <span>Drag handle</span>
</div>

<!-- Image cover in a fixed container -->
<div class="w-48 h-32 overflow-hidden rounded-lg">
  <img class="w-full h-full object-cover" src="photo.jpg" />
</div>

<!-- Vertically resizable textarea -->
<textarea class="resize-y w-full border rounded-md p-2"></textarea>

<!-- Remove native select appearance -->
<select class="appearance-none border rounded px-3 py-2">
  <option>Option 1</option>
</select>
```
