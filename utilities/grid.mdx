Grid utilities work in combination with `grid` or `inline-grid` from the [Layout](/utilities/layout#display) display utilities.

---

## Grid Columns

Define how many equal columns a grid container has with `grid-cols-{n}`.

| Class | CSS |
|-------|-----|
| `grid-cols-1` | `grid-template-columns: repeat(1, minmax(0, 1fr))` |
| `grid-cols-2` | `grid-template-columns: repeat(2, minmax(0, 1fr))` |
| `grid-cols-3` | `grid-template-columns: repeat(3, minmax(0, 1fr))` |
| `grid-cols-4` | `grid-template-columns: repeat(4, minmax(0, 1fr))` |
| `grid-cols-5` | `grid-template-columns: repeat(5, minmax(0, 1fr))` |
| `grid-cols-6` | `grid-template-columns: repeat(6, minmax(0, 1fr))` |
| `grid-cols-7` | `grid-template-columns: repeat(7, minmax(0, 1fr))` |
| `grid-cols-8` | `grid-template-columns: repeat(8, minmax(0, 1fr))` |
| `grid-cols-9` | `grid-template-columns: repeat(9, minmax(0, 1fr))` |
| `grid-cols-10` | `grid-template-columns: repeat(10, minmax(0, 1fr))` |
| `grid-cols-11` | `grid-template-columns: repeat(11, minmax(0, 1fr))` |
| `grid-cols-12` | `grid-template-columns: repeat(12, minmax(0, 1fr))` |
| `grid-cols-none` | `grid-template-columns: none` |

## Gap

Control the spacing between grid (and flex) items using the full [spacing scale](/utilities/spacing#spacing-scale).

| Prefix | CSS Property |
|--------|--------------|
| `gap-{key}` | `gap` |
| `gap-x-{key}` | `column-gap` |
| `gap-y-{key}` | `row-gap` |

```html
<!-- Uniform gap -->
<div class="grid grid-cols-3 gap-4">
  <div>Cell</div>
  <div>Cell</div>
  <div>Cell</div>
</div>

<!-- Different horizontal and vertical gaps -->
<div class="grid grid-cols-4 gap-x-6 gap-y-4">
  <div>Cell</div>
  <div>Cell</div>
  <div>Cell</div>
  <div>Cell</div>
</div>
```

## Common Patterns

```html
<!-- Responsive 3-column card grid -->
<div class="grid grid-cols-3 gap-6">
  <div class="bg-surface rounded-xl p-4 shadow">Card</div>
  <div class="bg-surface rounded-xl p-4 shadow">Card</div>
  <div class="bg-surface rounded-xl p-4 shadow">Card</div>
</div>

<!-- 12-column layout with spanning regions -->
<div class="grid grid-cols-12 gap-4">
  <aside class="col-span-3 bg-secondary rounded-lg p-4">Sidebar</aside>
  <main class="col-span-9 bg-surface rounded-lg p-4">Main content</main>
</div>

<!-- Full-width header spanning all columns -->
<div class="grid grid-cols-3 gap-4">
  <header class="col-span-full bg-primary p-4">Full-width header</header>
  <div>Col 1</div>
  <div>Col 2</div>
  <div>Col 3</div>
</div>

<!-- Centered tile using place-items -->
<div class="grid grid-cols-3 place-items-center h-48">
  <div>Centered tile</div>
</div>
```

<Tip>
  `gap-{key}` works on both grid and flex containers. No need to switch utilities when toggling between the two display modes.
</Tip>
