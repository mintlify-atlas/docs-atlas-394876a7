---
title: "Spacing tokens"
description: "The TSS spacing scale used by padding, margin, width, height, gap, and inset utilities."
---

The `SPACING` token is a flat key→value map used by all spacing utilities (`p-*`, `m-*`, `w-*`, `h-*`, `gap-*`, `inset-*`, etc.).
