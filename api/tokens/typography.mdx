## Font sizes (FONT_SIZE)

Each entry is a `[fontSize, lineHeight]` tuple. Use the `text-{key}` utility class to apply both values simultaneously.

| Key | Font size | Line height | Class |
|-----|-----------|-------------|-------|
| `xs` | 0.75rem | 1rem | `text-xs` |
| `sm` | 0.875rem | 1.25rem | `text-sm` |
| `base` | 1rem | 1.5rem | `text-base` |
| `lg` | 1.125rem | 1.75rem | `text-lg` |
| `xl` | 1.25rem | 1.75rem | `text-xl` |
| `2xl` | 1.5rem | 2rem | `text-2xl` |
| `3xl` | 1.875rem | 2.25rem | `text-3xl` |
| `4xl` | 2.25rem | 2.5rem | `text-4xl` |
| `5xl` | 3rem | 1 | `text-5xl` |
| `6xl` | 3.75rem | 1 | `text-6xl` |
| `7xl` | 4.5rem | 1 | `text-7xl` |
| `8xl` | 6rem | 1 | `text-8xl` |
| `9xl` | 8rem | 1 | `text-9xl` |

A line-height of `1` is unitless, meaning `line-height` equals the font size exactly (no additional leading).

## Font weights (FONT_WEIGHT)

| Key | Value | Class |
|-----|-------|-------|
| `thin` | 100 | `font-thin` |
| `extralight` | 200 | `font-extralight` |
| `light` | 300 | `font-light` |
| `normal` | 400 | `font-normal` |
| `medium` | 500 | `font-medium` |
| `semibold` | 600 | `font-semibold` |
| `bold` | 700 | `font-bold` |
| `extrabold` | 800 | `font-extrabold` |
| `black` | 900 | `font-black` |
